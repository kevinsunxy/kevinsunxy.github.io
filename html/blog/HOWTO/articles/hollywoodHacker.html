How to look like a Hacker in the Hollywood movie





Hackers in the movie are so cool, the stuff on their screen looks crazy, here is how to make your terminal look like theirs! All you need is an application called Hollywood. I only tried this on Ubuntu. It really made my terminal go nuts.



  1. Enter this command in your terminal to get the app

        sudo apt install hollywood


  2. When it is installed, enter the command

        hollywood


  3. To make it stop, hit ctrl+c (^c) twice, and enter the command

        exit




P.S. You may want to turn off the volume when you use this app, it plays mission impossible theme song as BGM... Have fun hacking! : )
