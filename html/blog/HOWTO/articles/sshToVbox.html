How to ssh to Ubuntu that runs in the VirtualBox

I'm new to Linux, and I was learning how to use ssh, I wanted to try the command, but I didn't want to login to the server of McGill, and unfortunately, that's the only server that I have access to. However, I had installed Ubuntu and Virtual Box, but can I log in to a virtual machine? If I can, how? Then I did some research, I succeeded and had a lot of fun with it, here is how:



  1. You need to go to Settings of the machine that you want to connect to, go to Network, set the attached to option to Bridged Adapter. This allows your VM to have a different IP(and some other stuff) than the host and be visible to other machines on the same network.


  2. Run Ubuntu, if you want someone else to be able to connect to this VM, you need to create a user account for this user.


  3. Install openssh server by typing this command in terminal

         sudo apt-get install openssh-server

  4. When it is done, type the command ifconfig and write down the IP address.


  5. In terminal, type

        sudo service ssh start

    and press enter to start the server. start can be replaced by stop, restart, status .


  6. In the terminal of the machine to which you want to connect the VM, type

        ssh username@IP

    The username is the account on Ubuntu, IP is the IP address that we find earlier. The password is the password for the account on Ubuntu.




This doesn't just work on VM, it also works on real computers, and you can login to a VM from another VM, etc.

That's it; you're in! Don't forget to close the server when you're done.
